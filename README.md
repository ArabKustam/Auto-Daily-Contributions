# GitHub Activity Bot

Этот скрипт автоматически создает коммиты, чтобы поддерживать вашу активность на GitHub (зеленые квадратики).

## Установка

1.  **Установите Git**: Убедитесь, что у вас установлен Git.
2.  **Установите Python**: Убедитесь, что у вас установлен Python.
3.  **Инициализация репозитория**:
    Если вы еще этого не сделали, откройте терминал в этой папке и выполните:
    ```bash
    git init
    # Замените URL на ваш репозиторий GitHub
    git remote add origin https://github.com/ВАШ_НИК/ВАШ_РЕПОЗИТОРИЙ.git
    git branch -M main
    ```
4.  **Первый запуск**:
    Запустите `run_bot.bat` вручную один раз, чтобы проверить, что коммиты создаются и отправляются.
    *Примечание: Вам может потребоваться настроить аутентификацию (SSH ключи или Git Credential Manager), чтобы `git push` работал без ввода пароля.*

## Автоматизация (Windows Task Scheduler)

Чтобы скрипт запускался каждый день автоматически:

1.  Нажмите `Win + R`, введите `taskschd.msc` и нажмите Enter.
2.  Справа выберите **Create Basic Task** (Создать простую задачу).
3.  **Name**: "GitHub Activity Bot".
4.  **Trigger**: **Daily** (Ежедневно).
5.  **Action**: **Start a program** (Запустить программу).
6.  **Program/script**: Выберите файл `run_bot.bat` в этой папке.
    * `C:\Users\MARK\Desktop\studing\my capcha soft\run_bot.bat`
7.  **Start in (Optional)**: Вставьте путь к папке:
    * `C:\Users\MARK\Desktop\studing\my capcha soft\`
    *(Это важно, чтобы скрипт нашел файлы)*
8.  Нажмите **Finish**.

Теперь ваш компьютер будет выполнять этот скрипт каждый день в указанное время.
